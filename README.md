# 🌍 Air Quality Prediction using Machine Learning

## 📌 Objective
The objective of this project is to predict the **Air Quality Index (AQI)** using machine learning techniques based on environmental pollutant data.  
This helps in understanding pollution patterns and making data-driven decisions for environmental monitoring.

---

## 📊 Dataset
The dataset contains air pollution parameters such as:
- PM2.5
- PM10
- NO, NO2, NOx
- NH3, CO, SO2
- O3
- Benzene, Toluene, Xylene

### 🔧 Data Preprocessing:
- Handled missing values  
- Feature selection and correlation analysis  
- Data splitting (train-test)  
- Feature scaling (where required)  

---

## 🤖 Models Used
The following machine learning models were implemented and compared:

- Linear Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  
- XGBoost ⭐ *(Best Performing Model)*  

### ⚙️ Techniques Applied:
- Cross-validation  
- Hyperparameter tuning  
- Model comparison  

---

## 📈 Results
- Models were evaluated using:
  - Mean Absolute Error (MAE)  
  - Mean Squared Error (MSE)  
  - R² Score  

- ✅ **XGBoost achieved the best performance** among all models with highest accuracy and better generalization.
